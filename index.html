💡 本项目学习要点

  • 主进程: main.js - 负责创建窗口和管理应用生命周期
  • 渲染进程: index.html + renderer.js - 负责 UI 渲染和用户交互
  • 预加载脚本: preload.js - 安全地暴露 Node.js API
  • 进程通信: 使用 contextBridge 和 IPC 进行安全通信
